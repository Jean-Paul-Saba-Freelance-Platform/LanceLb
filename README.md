# Freelance Project

A platform to help people find freelance jobs.

## Project Structure

```
FreelanceProject/
├── backend/          # Backend server (Node.js/Express)
│   ├── server.js
│   └── package.json
└── frontend/         # Frontend React app (Vite)
    ├── src/
    ├── pages/
    └── package.json
```

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Features

- Authentication (Login/Signup)
- Glassmorphism UI design
- User types: Freelancer and Client/Employer
- Landing page + Routing to sign in/ log in
